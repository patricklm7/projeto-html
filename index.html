Dos primeiros computadores até atualidade

Não se pode falar da internet sem falar antes sobre a história dos computadores,

que sofreu uma imensa evolução desde dos computadores á válvula, depois dos primeiros microprocessadores,

a chegada no computador pessoal mais compacto e hoje com vários dispositivos que tem funções parecidas ou similares a computadores de mesa.

E hoje em dia essa evolução se deu através da conectividade,

a interligação de um computador para outro computador, que se evolui e se transformou em uma teia complexa,

que é capaz de conectar inúmeros usuários de forma simutânea, ligando todo o mundo.

O que são Clients?

Os usuários que são que usufruem desses serviços são chamados de Clients .

Os Clients comumente faz a solicitação através de um navegador de que pagina quer acessar,

E automaticamente o navegador traduz em uma interface uma página em html, digamos que o Client seja algo pacivo, é nele que chegara os serviços.

Mas, não podemos descartar outras formas que o Client pode ser servido, como celulares, smartwhatchs, Televisores até mesmo geladeiras conectadas a rede,

é claro que nesses casos não é comum usar a estrutura HTML

O que são Servers?

As informações que são enviadas aos Clients vem de lugares onde são servidas essas informações, no caso os Servers ,

As informações são guardadas e geridas em banco de dados, quando a uma solicitação de algum usuário, deve haver alguma resposta do servidor.

Essa conexão pode ser simples e fechada com uma Intranet uma rede fechada com poucas máquinas, ou mais complexa como a Internet, onde o usuário comum pode acessar vários serviços

Como é uma estrutura básica de um HTML?

HTML Abreviatura de HiperText Markup language é um exemplo de linguagem de marcação muito ultilizada para o desenvolvimento de paginas Web,

é utilizada tags para definir sua estrutura exemplo a tag body que define o corpo do texto, ou a tag head que define o cabeçalho
